## Referencia

Materiales de referencia rápida para narradores.

## Qué hay en esta sección

### [6.1 Configuración](/guia/referencia/configuracion/)
Ajustes del sitio y opciones en `_config.yml`.

### [6.2 Referencia CSV: proyecto y objetos](/guia/referencia/csv-proyecto-objetos/)
Referencia de columnas para project.csv y objects.csv, incluyendo soporte bilingüe.

### [6.3 Referencia CSV: historias y glosario](/guia/referencia/csv-historias-glosario/)
Referencia de columnas para CSVs de historias y glossary.csv.

### [6.4 Sintaxis de Markdown](/guia/referencia/sintaxis-markdown/)
Referencia completa para todo el formato markdown disponible en paneles y páginas de Telar.

### [6.5 Validador de Configuración](/guia/referencia/validador-de-configuracion/)
Pega tu `_config.yml` para buscar errores.

### [6.6 Generador y editor de configuración](/guia/referencia/generador-de-configuracion/)
Crea un nuevo `_config.yml` o carga y edita uno existente.

### [6.7 Referencia de Google Sheets](/guia/referencia/google-sheets/)
Configuración avanzada de la hoja de cálculo, solución de problemas y pestañas.

### [6.8 Insertar en Canvas](/guia/referencia/insercion/)
Inserta tu sitio Telar en un sistema de gestión de aprendizaje.
