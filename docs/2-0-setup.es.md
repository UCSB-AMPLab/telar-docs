# Configura tu sitio

Elige un método de configuración según tus necesidades y nivel de comodidad técnica.

## Métodos de configuración

### Configuración manual — Google Sheets + GitHub

Construye una exposición completa desde tu navegador. No necesitas instalar nada — solo una cuenta de Google y una cuenta de GitHub. Este es el enfoque manual paso a paso para personas que ya se sienten cómodas con GitHub y YAML.

[Configuración manual](/guia/configuracion/configuracion-manual/){: .btn .btn-primary }

{: .tip }
> **¿Sin experiencia con GitHub?** El [Inicio rápido](/guia/primeros-pasos/inicio-rapido/) es un punto de partida más fácil — te guía de manera interactiva y genera tu archivo de configuración automáticamente.

### Desarrollo local — CSV + Jekyll

Trabaja completamente en tu computadora con archivos CSV y un servidor Jekyll local. Sin Google Sheets, sin GitHub — tú controlas todo.

Ideal para cursos de humanidades digitales, personas con experiencia técnica, y cualquiera que quiera enseñar o aprender control de versiones, sitios estáticos y datos estructurados mientras construye su exposición.

[Configurar localmente](/guia/configuracion/desarrollo-local/)
