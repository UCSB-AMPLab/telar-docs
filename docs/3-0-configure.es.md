# Configura tu proyecto

El comportamiento de Telar se controla a través de `_config.yml`, un archivo YAML en la raíz de tu repositorio. Allí se definen el título de tu sitio, el tema, el idioma, la conexión con Google Sheets y las opciones de funciones.

Si usaste el [Inicio rápido](/guia/primeros-pasos/inicio-rapido/), ya tienes un `_config.yml` funcional. Esta sección te ayuda a entender qué contiene y cómo modificarlo.

## En esta sección

- [Generador y editor de configuración](/guia/configurar/generador-de-configuracion/) — Crea un nuevo `_config.yml` desde cero o carga uno existente para editarlo
- [Referencia de configuración](/guia/configurar/configuracion/) — Lista completa de cada opción y qué hace
- [Validador de configuración](/guia/configurar/validador-de-configuracion/) — Pega tu `_config.yml` para verificar errores antes de hacer commit
