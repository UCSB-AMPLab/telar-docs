# Funciones del sitio

Telar incluye varias funciones integradas que enriquecen tu exhibición más allá de la experiencia básica de las historias.

- [Galería de objetos](/guia/funciones/galeria-objetos/) — Una página de galería navegable y con búsqueda para toda tu colección, con filtrado por tipo, creador, periodo y temas
- [Glosario](/guia/funciones/glosario/) — Define términos que el público puede consultar desde las historias sin salir de la narrativa
- [Historias privadas](/guia/funciones/historias-privadas/) — Encripta historias para que solo quienes tengan la clave correcta puedan acceder a ellas
- [Páginas personalizadas](/guia/funciones/paginas-personalizadas/) — Crea páginas independientes para contenido fuera de la estructura de historias — créditos, metodología, bibliografías
- [Insertar en Canvas](/guia/funciones/insercion/) — Inserta tu sitio Telar o historias individuales en sistemas de gestión de aprendizaje
